# 🧮 Калькулятор відкатів

Офлайн-калькулятор для розрахунку ціни товарів/послуг з урахуванням податків, зборів та комісій. Створено виключно в навчальних цілях та з спортивного інтересу — як вправу з фінансової математики та податкового законодавства України.

## Що вміє

- **Два режими**: «Відкат → Ціна» (скільки виставити в договорі) та «Ціна → Відкат» (скільки вийде чистими)
- **Системи оподаткування**: ФОП 2 група, ФОП 3 група (5% та 3%+ПДВ), ТОВ з ПДВ та без, довільні параметри
- **Податки та збори**: єдиний податок, ПДВ 20%, військовий збір 5%, комісія переказу
- **Деталізація**: повна розбивка по кожному рядку — брутто, податки, чисті на руки, прибуток виконавця
- **Собівартість**: опціональне поле для розрахунку реального прибутку

## Запуск

Один HTML-файл, нуль залежностей. Відкрити `index.html` у браузері — і все.

Для деплою на GitHub Pages — покласти файл у репозиторій і увімкнути Pages.

## Приватність

Калькулятор працює повністю на стороні клієнта. Жодного логування, аналітики, cookies, зовнішніх запитів чи збору даних не ведеться. Все рахується локально в браузері. Працює офлайн.

## Дисклеймер

Проєкт створено суто з спортивного інтересу як вправу з арифметики та верстки. Автор не несе відповідальності за будь-яке використання цього інструменту. Калькулятор має виключно інформаційний характер і не є фінансовою чи юридичною консультацією. Ставки податків актуальні на 2025–2026 рр. — звіряйтесь з чинним законодавством.
